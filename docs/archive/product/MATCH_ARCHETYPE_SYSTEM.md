# MATCH ARCHETYPE SYSTEM

**Product core:** social **energy compatibility** (rhythm + cost), not dating score, personality type, therapy, or astrology.

Implementation: `product/match/runtime/match_rhythm_engine.py`

---

## stable_low_pressure

| Field | Content |
|-------|---------|
| **Behavior** | Predictable replies; low emotional spikes; silence rarely used as pressure. |
| **Rhythm** | Slow-stable; gaps are normal. |
| **Energy cost** | Low |
| **Ideal pairing** | Users with high reply sensitivity who need async-safe windows. |
| **Dangerous pairing** | Users who need constant validation bursts. |
| **Fatigue pattern** | Lengthy chats tire you; rarely emotional weight. |
