# UAT EXPERIENCE CONSTITUTION (PHASE1)

Official UX constitution for **PHASE1** human UAT and future UI/copy work.  
Complements [`ops/product/PRODUCT_CONSTITUTION.md`](PRODUCT_CONSTITUTION.md).

---

## Official PHASE1 user journey

1. **Home** — Guardian positioning: emotional safety, signal filtering, calm observation.  
2. **Profile (`/profile`)** — Build a **signal file**: habits, interests, stress rhythm the fox uses for observation.  
3. **Observation room (`/insight`)** — Run a **guardian observation** (demo flow): scan → result → explanation → next step.  
4. **Repeat** — User may re-observe, update their signal file, or review notes/history.

---

## Emotional goals

* Reduce anxiety about “what is this product?”
* Reduce shame about “am I doing it wrong?”
* Increase sense of **protection** and **clarity**
* Keep pressure **low**; language **quiet**

---

## Guardian behavior rules (UX)

* The fox **observes** and **warns**; it does not judge the user’s worth.
* Copy must be **explain-first**: what happened → why it matters → what to do next.
* Prefer **signal / boundary / rhythm** language over relationship scoring language.

---

## Onboarding philosophy

* **No mystery doors**: every screen answers “why am I here?” in one short beat.
* **One primary next action** after each milestone (save profile → go observe).
* **No dead ends** after a result: always show next actions.

---

## Forbidden UX patterns

* Dense “dashboard” layouts without narrative context
* Hidden prerequisites (e.g., profile purpose unexplained)
* Post-result screens with no next step
* Alarmist visuals (harsh red alert storms) for normal caution states

---

## Forbidden dating / romance language

* dating / romance / soulmate / chemistry framing
* “matching” as the product identity
* competitive ranking / scoreboard metaphors as primary framing

---

## Explanation-first UX principle

Every non-trivial output must be paired with:

1. **What** the user is seeing (plain language)  
2. **Why** it showed up (guardian observation, not moral judgment)  
3. **What to do next** (bounded, calm actions)

---

## Fox role definition

The Arctic Fox is a **guardian observer**:

* filters noisy / draining signals
* names patterns that repeat
* recommends distance and pacing when risk is elevated

The fox is **not**:

* a dating coach
* a compatibility rater
* an engagement-maximization engine
