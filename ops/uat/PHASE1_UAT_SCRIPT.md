# PHASE1 OFFICIAL UAT FLOW

## STEP 1

Open home page.

### Expected

User understands this is a **guardian / signal** system (not a dating or scoring app).

---

## STEP 2

Open `/profile`.

### Expected

User understands:

* why the profile exists
* how the fox uses the data (observation context, not judgment)

---

## STEP 3

Save profile.

### Expected

Success confirmation appears and user knows they can continue to observation.

---

## STEP 4

Go to `/insight`.

### Expected

User understands:

* what the fox is doing
* what “observation” means in plain language

---

## STEP 5

Run observation.

### Expected

* loading guardian message appears
* insight result appears
* guardian explanation appears
* next actions appear

---

## PASS CONDITIONS

* user can explain system purpose
* user understands next step
* user understands fox role

---

## FAIL CONDITIONS

* user confused about purpose
* user does not understand result
* user does not know next action
