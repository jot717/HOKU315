# GUARDIAN UX CONSTITUTION (PERMANENT)

This file is **binding UX law** for HOKU315 **FOX GUARDIAN SYSTEM** product experiences.  
It complements [`PRODUCT_CONSTITUTION.md`](PRODUCT_CONSTITUTION.md) and [`UAT_EXPERIENCE_CONSTITUTION.md`](UAT_EXPERIENCE_CONSTITUTION.md).

---

## 1. PRODUCT IDENTITY

HOKU315 is:

* a **guardian world** (not a generic AI tool surface)
* a **signal protection system** (filter noise, pressure, and draining patterns)
* an **emotional safety layer** (reduce exposure before harm compounds)
* an **anti-pressure assistant** (pace, distance, clarity over intensity)

The product is **not**:

* a dating product
* a personality scoring lab
* a “smart matching” showcase

---

## 2. FORBIDDEN UX PATTERNS

Forbidden:

* compatibility language
* emotional resonance analysis (as a product frame)
* personality scoring
* psychological report style
* AI analysis wording (“report”, “analysis engine”, “model output” as the headline identity)
* dating app feeling
* “matching system” framing
* long analytical paragraphs as the first screen
* score-first UX (numbers before meaning)

---

## 3. REQUIRED UX RULES

Must:

* **conclusion first** (what matters now)
* **danger first** (what could drain the user)
* **guardian tone** (protective, calm, observational)
* **fox presence visible** (the guardian is a character, not a logo)
* **short emotional guidance** (small number of lines; breathing room)
* **explain WHY clearly** (plain language; max a few bullets)
* **always tell the user what to do next** (never a “dead end” screen)

---

## 4. UX HIERARCHY (CORRECT ORDER)

1. Fox appears (presence + immediate emotional guidance)
2. Fox warning / risk headline (single clear statement)
3. What danger exists (short, concrete)
4. Why it matters (short bullets)
5. What the user should do (guardian guidance)
6. Optional deeper notes (collapsed / secondary only if needed)

---

## 5. UX FEELING (TARGET)

Users should feel:

* warmth
* safety
* emotional relief
* protected
* low pressure
* guided

Users must not feel:

* analyzed
* tested
* judged
* emotionally exposed
