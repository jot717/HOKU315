# PHASE1 — Product flow (locked)

**Status:** ACTIVE — v1  
**Scope:** Single primary user journey (SAFE MODE: copy, navigation, and UX structure only).

## End-to-end flow

1. **HOME** `/` — What the product is, five numbered steps, CTAs to profile and quiz.  
2. **PROFILE** `/profile` — Create *my* signal profile (name, interests, stress).  
3. **QUIZ** `/quiz` — Twenty slider questions refine interaction sensitivity (not a second profile).  
4. **TARGET** `/target` — Describe the person or context under observation.  
5. **INSIGHT** `/insight` — Run analysis: interaction risk, drain, manipulation signals, rhythm, next actions.  
6. **MATCH** `/match` — Signal-filtered discovery; high-risk combinations de-emphasized or blurred per existing rules.

## Product identity (one sentence)

**AI-native social signal intelligence:** interpret dangerous interaction patterns, emotional drain risk, and social rhythm — not therapy, not a fox universe, not a generic personality test.

## Fox layer

Arctic fox copy is **secondary**: short reminder after system analysis, not stacked guardian cards or parallel onboarding systems.
